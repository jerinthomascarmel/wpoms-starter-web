# Project tech : 
- project is made with react js
- completely written with vanilla css in module.css (for better decoupling) 
- use lucid icons for getting icons if not present do your own way.

# project structure 
project have folders in src such as : 
/components
/pages
/services
/routes
/utils

/pages contains page components like- register page , login page , dashboard 
/services contains all the apis with backend. 
/routes contains routing using react router dom (Browser router)
